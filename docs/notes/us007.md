# Story [[display-platform]] [[product]]

## role
[[pn001|developer]]

## want
work on my development machine remotely — reach its full graphical desktop from another device

## because
the dev environment (WSL, tools, configs) lives on one host and I need to use it from elsewhere without replicating the setup

## acceptance_criteria
- connect from a remote client and land in an interactive graphical desktop session of the development machine
- the desktop is reachable with no monitor attached to the host (headless)
- the session returns after a host reboot without an interactive local login first
- terminal, editor and file manager in the session respond to keyboard and pointer over the link
