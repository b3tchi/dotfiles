# Story [[project-config]] [[navigation]] [[product]]

## role
[[pn001|developer]]

## want
register a particular filesystem location as a project so i can easily navigate to it later without remembering the full path

## because
i work across many projects scattered in different directories; remembering and typing full paths every time is friction, and having a known list of projects lets me jump in by name

## acceptance_criteria
- can register a filesystem location as a project with a short identifier
- registered projects are listed somewhere i can browse or pick from
- can jump to a registered project by its identifier from anywhere on the system
- registration persists across sessions
