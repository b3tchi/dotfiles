# Story [[remote-access]] [[security]] [[product]]

## role
[[pn001|developer]]

## want
access and work on my workstation remotely, with my full graphical
desktop environment

## because
I need my complete workstation — editor, shells, tools, windows — from
other devices like my phone

## acceptance_criteria
- a remote connection lands in a full graphical desktop session,
  independent of and not disturbing the local console session
- the machine is reachable for remote access only from trusted networks,
  never from the open internet
- credentials and session traffic are encrypted with a connection-specific
  secret, not a secret shared across installations
- only explicitly permitted accounts can establish a remote session, and
  repeated failed login attempts are throttled
- the remote session is usable as a primary work environment without
  degrading or interfering with the local session
