## summary
Decisions about the graphical session stack: display servers (X11/Wayland), window managers, remote-desktop/remoting tech (RDP, VNC), GPU acceleration, and host display bridges (WSLg, Termux:X11).

In: WM choice per platform, display-server selection, remoting protocols and servers, GPU/driver plumbing (d3d12, DRM), compositor backends. Out: terminal emulators and TUI stack (core UX, not display layer); workflow scripts and repo tooling (cat001 workflow-tooling).
