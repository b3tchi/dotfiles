# Feature [[cat001]] [[product]]

## providing
Vault-backed secret retrieval and rotation. Every service calls
`secret(name)` to read credentials at runtime. Operators call
`rotate_secret(name, new_value)` to stage a new versioned alias
without service downtime.

## api_surface
```python
from acme.lib.vault import secret, rotate_secret, VaultError
db_url = secret("reports/db_url")
rotate_secret("reports/db_url", "<new-credential>")  # per-name lock
```

## data_model
None local. Vault is the source of truth.

## sample
`src/lib/vault.py` ships the canonical client.

- `src/lib/vault.py`
