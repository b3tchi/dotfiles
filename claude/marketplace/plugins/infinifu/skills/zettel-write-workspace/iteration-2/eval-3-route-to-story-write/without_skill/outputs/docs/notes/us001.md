# Story [[product]]

## role
[[pn001|field-rep]]

## want
upload a csv of contacts in bulk

## because
typing entries one at a time in the office after a day in the field wastes evenings; a bulk import frees up that time and reduces data-entry fatigue.

## acceptance_criteria
- field rep can select a local csv file and submit it as a single upload
- the system parses the csv, validates rows, and creates contact records for valid rows in one operation
- invalid rows are reported back with row number and reason; valid rows are still imported
- a summary of created / skipped / failed counts is shown after upload
- priority: medium
