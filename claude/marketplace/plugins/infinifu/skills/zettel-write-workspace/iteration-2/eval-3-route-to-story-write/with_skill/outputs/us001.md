# Story [[product]]

## role
[[pn001|field-rep]]

## want
upload a csv of contacts in bulk instead of entering them one by one

## because
field reps lose evenings re-typing entries at the office that they already collected during the day

## acceptance_criteria
- accept a .csv upload from the contacts screen
- preview parsed rows before commit, with row-level error messages for bad rows
- commit only the valid rows; reject the bad ones without aborting the whole upload
- imported contacts appear in the contact list with the same fields as manually-created ones
