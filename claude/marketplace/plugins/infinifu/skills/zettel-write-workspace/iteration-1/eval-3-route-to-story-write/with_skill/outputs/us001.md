# Story [[product]]

## role
[[pn001|field-rep]]

## want
bulk-upload contacts from a CSV file in one operation

## because
typing entries one at a time in the office wastes evenings that should be field time

## acceptance_criteria
- accept a CSV file upload as the entry point for many contacts at once
- preview parsed rows before commit so the rep can spot mistakes early
- reject malformed rows with row-level error messages instead of failing the whole batch
- successful import creates one contact per valid row in a single operation
