# Story [[platform-flow]] [[product]]

## role
[[pn002|platform-engineer]]

## want
the 5-minute overlap timer, automatic expiry of the prior credential,
and the synthetic-check hook for the rotation window all run without
manual operator intervention

## because
[[sp001]] shipped the `vault.rotate_secret` write path only — the
orchestration module (`vault_rotate.py`) and the integration synthetic
check that [[us003]] acceptance criteria assume were scoped as Tasks
2 and 3 of the spec but no code landed for them; without this the
"no 5xx during the rotation window" guarantee is unverified in
production

## acceptance_criteria
- An orchestration entry point schedules expiry of the prior credential
  at T+5min using a monotonic clock
- Pending expiries resume on scheduler restart
- A synthetic check runs every 30s during the rotation window and
  fails the rotation if a 5xx is observed (tolerating a single
  transient blip)
