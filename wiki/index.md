## How to use

- New note → drop in `notes/` (or use markdown-oxide code action on unresolved link)
- Wikilinks: `[[note-name]]` or `[[note-name#Heading]]`
- Tags: `#topic`
- Daily notes: `daily/YYYY-MM-DD.md`
- markdown-oxide indexes whole vault, gives completion + backlinks via LSP
